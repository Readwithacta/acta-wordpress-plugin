# Acta — WordPress Plugin

Monetize your content with a seamless paywall. Readers pay per article using cards, Apple Pay, or Google Pay — in seconds.

## Requirements

- WordPress 5.8+
- PHP 7.4+

## Installation

Install via the WordPress admin or upload to `/wp-content/plugins/acta-content/`, then follow the on-screen setup to connect your Acta account and complete Stripe onboarding.
